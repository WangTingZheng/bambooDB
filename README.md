## BambooDB

add module

```shell
git clone https://github.com/WangTingZheng/bambooDB.git
git submodule init
git submodule update
```

build and run test

```shell
cd bambooDB/test
./build.sh
./run
```

build and run benchmarks

```shell
cd bambooDB/bench
./build.sh
./test
```
